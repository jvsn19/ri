What Is This Game?

The Forest's Entwine is  a text based choose your own adventure game where you play as an investigator searching for a missing person in a strange town. The game is heavily inspired by and seeking to replicate lovecraftian horror elements and themes with its musical score and writing style.

The game is as much as I was able to do in the short amount of time I was given to work on this project (less than two months). I am hoping to polish up the writing and programming chunk by chunk over time.  The end goal for this game is to get it working with the Loominary system in order to make the color system more visual to players as the game progresses, but it can be played with a mouse just as easily.

Can It Run On...

The game is an html file, just unzip it and it can run in any browser, even offline. You need a mouse, a working monitor, and speakers/headphones.

Who are you?

I'm Adam Cohen and I made this game in a bit under 2 months as a solo project for UCF's Fall Game Design Workshop 2017 class. I am a writer, online personality, and designer. I basically live off of coffee.
