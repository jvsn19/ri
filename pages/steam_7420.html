IMPORTANT NOTICE: DLC Price Changes

Following this sale, we'll be increasing the base price of all of our current DLC Kits (MMORPG, 3D FPS / Survival Horror and Point & Click Adventure). The reason for this is because we plan on introducing new asset DLC in the future and realised that the template DLC we currently provide is undercut considerably, when you stop to consider just how much content is provided. We also didn't factor in just how much on-going support work would be involved.

If you want to grab these DLC Kits at their lowest price, then now is the time to do so!

NEW Version: v1.020.002

We've also updated 001 Game Creator to v1.020.002. This update mostly consists of bug fixes and minor additions (some of which we've highlighted below). You can see the full change log here: https://www.engine001.com/changes.htm

NEW "Used Item" Trigger - a new Actor trigger that executes whenever an Actor uses any Item, with optional Equipment Area and Item filters. This trigger can take advantage of our Item Constants feature that we added in the last version! See our previous announcement for details on how Item Constants work.



NEW Audio Conversion - right-clicking an audio file (from within the Sounds and Music window) and selecting "Effects" will bring up a new audio conversion window that allows users to convert any audio to OGG or WV format. This is particularly useful when developing HTML5 and mobile games where MP3/MIDI files aren't supported.



UPDATED JDK for Android - we've made Android building work with JDK 8, instead of relying on an outdated (and no longer supported) version.

We've also updated Android building to target Android 8.0.0 (API Level 26) to coincide with new Google Play requirements effective August 2018.

Until Next Time...

We hope you enjoy these version changes and are understanding for our reasons for increasing DLC Kit prices. We have plenty of new content just over the horizon and we look forward to sharing details of those with you soon!

31 May

MMORPG Tutorials and New Version

At long last the MMORPG video tutorial series is finished and available for your viewing pleasure. We've also updated 001 with a bunch of cool additions that we've detailed below.

MMORPG Video Tutorials



In this four-part series, you'll learn how online multiplayer games work, how to properly setup your server, how to create different types of NPC Actors, how to create Item/Kill Quests as well as how to create console commands and build your game.

You can watch the full series on our official YouTube channel here: https://www.youtube.com/playlist?list=PLgohPz-X6YwhOamgHq95xH0OQjiZp36d4

Since this series relies heavily on the MMORPG DLC Kit, you'll also need to purchase and install this if you haven't done so already.

https://store.steampowered.com/app/564680/001_Game_Creator__MMORPG_Kit/

NEW Version (v1.020.001)

A new version of 001 Game Creator is now available, with a host of bug fixes and cool additions, some of which have been detailed below:

- NEW "Billboard (3D)" Tile Shape - This is a fairly big addition as it makes isometric games considerably simpler to build and 2D billboards in 3D games will no longer require Actors. Before, users would be required to use Actors for basically everything in 3D/isometric projects (which isn't very performance efficient). Thanks to this new addition this is no longer the case!



We've also made it possible to rotate the camera vertically in isometric projects and made it possible to use the "Save 3D Perspective" Map option for these projects as well.

- NEW Item/Magic Resource Constants - These are similar to Variables and are used to define special values for Items/Magic. Like with other properties of resources, these values cannot be changed through scripting but these values may be retrieved through scripting ("Constant" and "Constants Collection" Item/Magic use values). These are super useful when doing things like status effects, which will be a part of our upcoming Turn-Based Battle DLC Kit.



As per usual, you can find the full list of changes here: https://www.engine001.com/changes.htm

Until Next Time...

We hope you enjoy these new additions and also our MMORPG video tutorial series. We're always listening to community feedback on how we can improve the engine further and what we should be working on next, so please feel free to continue making suggestions that you think will benefit 001 and the 001 community!

About This Software

Create games without limits
001 Game Creator prides itself on offering users a diverse set of tools and features to create any type of game you can think of:

  • PC, Steam, Android/iOS (native), HTML5 with multi-finger gestures and network testing
  • Online Multiplayer/LAN/Networking (supports both TCP and UDP)
  • Multiple layer map editor with parallax canvases
  • 2D/3D camera system with 7.1 3D HD audio
  • Split-screen games with multiple gamepad support
  • Sprite layer, 3D model, normal mapping, custom shader materials and particle effect systems
  • Dynamic resource loading/unloading to improve performance/reduce memory usage for larger games
  • Interface/HUD designer
  • Built-in graphics editor
  • AI action battle mechanics and actor routes
  • Choice of graphical script and text code editors with game debugger, variable watching and statistics

Graphically Assisted Scripting
001 Game Creator offers creators the ability to craft intuitive and imaginative games without any prior coding knowledge. This is especially beneficial to artists and designers who prefer a more hands-on and visualised approach or developers who don’t wish to spend hours bashing their head against a wall. Unlike traditional Event-Driven Programming applications, 001 Game Creator offers a huge set of built-in events, with the ability to also create custom events for specific tailor-made functionality just for your games.

PC, Steam HTML5 & mobile development all-in-one
001 Game Creator allows developers to build their games for Windows PCs (exe) - with Steam API support (achievements, statistics, etc.), HTML5 (js/html) and mobile devices (apk) all-in-one! Unlike other similar engines, 001 Game Creator provides developers with the freedom to create their games for all available platforms without paying anything extra. Best of all, there are no additional license or revenue fees for selling your games - so if you’ve purchased a copy of 001 Game Creator you can sell your games without having to share the profits with us!

Play games with everyone
001 Game Creator gives developers the tools to create ambitious online multiplayer/co-op experiences with built-in networking features (TCP/IP). In addition, 001 Game Creator also allows developers to create local multiplayer experiences with up to four players - with multiple gamepad support included.

The possibilities are endless
001 Game Creator unlocks the doors to a wide range of possibilities. Make any game you can think of, including (but not limited to) RPGs, shoot-‘em-ups, racing games, fighting games, simulation games, platformers, tower defense, card games and even 3D games! If you can imagine it, you can create it with 001 Game Creator.

Please note: Our documentation is currently only available in English at this time.
