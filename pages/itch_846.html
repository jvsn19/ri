HexaPong

Support This Alexius.rait@gmx.at

A pong game for 3 to 10 players using smartphones as controllers. No apps need to be installed, both the client and the host are running in javascript. 

Client has to go to jackofall.games/pong

Min amount of players: 3


Browser support:

Yes: Firefox, Chrome
Should work: Opera, Edge, Safari

Use Firefox/Chrome on your mobile device for best experience, also lock the screen rotation if possible.
