    The Castle Far Cry 3 Blood Dragon set is now available. Make Castle embody Sergeant Rex “power” Colt with the FC3 Blood Dragon headgear, UMP45 Blood Dragon weapon skin, and charm.

    • Windows

    Minimum

    • OS: Windows 7, Windows 8.1, Windows 10 (64bit)
    • Processor: Intel Core i3 560 @ 3.3 GHz / AMD Phenom II X4 945 @ 3.0 GHz
    • Memory: 6 GB
    • Graphics: NVIDIA GeForce GTX 460 / AMD Radeon HD 5870 (DirectX-11 - 1GB of VRAM)

    • DirectX: 9.0c
