AAAAA!!!

A downloadable game for Windows, macOS, and Linux

STORY

"Congratulations! Your mad science endeavors and binge-watching horror movies has led you to create life. Unfortunately, you were watching The Blob, and now your creation is tearing through your lab. Luckily you have your trusty jetpack - can you escape in time?!"

CONTROLS

"AAAAA!!!" is a combination shooting/flying game created for the Game Maker's Tool Kit Game Jam 2017. It controls by clicking/holding the left mouse button. Please enjoy!

Install instructions

"AAAAA!!!" is an executable .JAR file that can run on Windows, Apple, and Linux OS. In order to run it, you must have Java installed on your computer. From there, it should be as simple as double clicking on the program; if that doesn't work, please look up how to run a .JAR executable on your operating system.
