Stryd

A downloadable game for Windows

Stryd is a card-collecting game based on Time Baroons.
All the cards are available to you from the very beginnig though. Because everyone hates grinding, right?

Multiplayer is implemented using Steam API, so make sure Steam is running before attempting to host or join.

Decks are stored in AppData\Local\Stryd\decks.txt - in case you're willing to backup / share them.

The game's currently in a very early state, so feel free to suggest / criticise things. Let's design the perfect meta together!
