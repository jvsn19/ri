Legacy Code

A downloadable game for Windows

Legacy Code is an action roguelite platformer.  Smash drones,  beat up (evil mutated) forest animals, and cover the walls with blood!

IF FULLSCREEN DOESN'T WORK, RUN IT IN WINDOWS 8 COMPATIBILITY MODE (thanks windows 10)

INSTALL THIS DIRECTX THING IF IT CRASHES WITH SOMETHING LIKE:

############################################################################################

Thanks to everyone who played the last demo and gave feedback,  especially to those who made videos, that kind of stuff is super useful.

DD16 Update:

  • Vending machine- Up to interact, arrows to select, jump to buy
  • Main menu and pause menu
  • Rebindable controls
  • Simplified shooting controls, you can now shoot while moving

Default controls:

  • Arrow keys - Move
  • Up - Pickup / Interact
  • Z - Attack
  • X - Jump
  • C -Shoot 
  • Z + Down (on the ground)- Slide kick
  • Z + Down (in the air) - Dive Kick
  • Alt+Enter - Fullscreen
  • Escape - Menu
