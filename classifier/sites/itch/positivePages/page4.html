3DNes Overview

- Play NES games with an unique 3D experience

- Support any display resolution - window mode or full screen

- Support game speed adjustment

- Support Save/Load states

- Support zipped roms

- Create your own 3D interpretation.

- Share/Get 3D customizations at the 3DN Repository

-  [3DNesVR] Play with Virtual/Mixed Reality Headsets Via Oculus or SteamVR

-  [3DNesVR] Retina Rendering Mode

-  [3DNesVR] Character Tracking Camera  - Pro Feature

Quick Guide

- 3dn files are located in the sub folder ./3dn

-  Adjust 3dn and rom filename so that they match each other - if you already have the mario.nes rom then change the 3dn file name to mario.3dn 

- If the corresponding 3dn file doesn't exit that's fine and the rom still works

- Launch 3dnes, open rom file then enjoy it yourself!
